## Tips for Taking Screenshots

- Use a clean terminal with good contrast
- Ensure text is readable (minimum 14pt font recommended)
- Capture actual usage, not mock data
- Show real features in action
- Keep screenshots up-to-date with current UI

## Image Format

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1200px width for clarity
- **File size**: Try to keep under 500KB per image
- **Color**: Full color, good contrast for readability
